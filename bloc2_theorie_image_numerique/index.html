    Bloc 2 : De l'image captée à l'image travaillée

    Théorie - Image numérique, retouche et typographie

    1. Comprendre l'image numérique

    Après avoir capté vos images sur le terrain en appliquant les principes de composition, lumière et couleur, il est temps de comprendre leur nature technique pour mieux les travailler.

    1.1 Pixels et résolution

    Qu'est-ce qu'un pixel? Le pixel (picture element) est la plus petite unité d'une image numérique. C'est un carré de couleur uniforme. Une image numérique est composée de milliers, voire de millions de pixels assemblés comme une mosaïque.

    Résolution d'image La résolution désigne le nombre de pixels contenus dans une image, exprimée en largeur × hauteur. - Exemple : 1920 × 1080 pixels = 2 073 600 pixels (environ 2 mégapixels) - Plus il y a de pixels, plus l'image peut être affichée en grand sans perte de qualité

    Résolution d'affichage (PPI/DPI) - PPI (Pixels Per Inch) : nombre de pixels par pouce pour les écrans - DPI (Dots Per Inch) : nombre de points par pouce pour l'impression - 72-96 PPI : standard pour l'affichage écran (web, réseaux sociaux) - 300 DPI : standard pour l'impression de qualité

    Règle pratique - Pour le web : privilégiez des dimensions adaptées (1080px de large pour Instagram, 1200px pour Facebook) - Pour l'impression : calculez vos dimensions en fonction de la taille finale (ex: 10cm × 15cm à 300 DPI = 1181 × 1772 pixels)

    1.2 Définition vs Résolution

    Définition Nombre total de pixels dans l'image (largeur × hauteur) - Une image de 4000 × 3000 pixels a une définition de 12 mégapixels

    Résolution Densité de pixels dans un espace donné (PPI/DPI) - La même image de 12 mégapixels peut avoir différentes résolutions selon son usage

    Exemple concret Une photo de 3000 × 2000 pixels : - À 72 PPI : fera 106 × 71 cm à l'écran (basse densité, pixels visibles de près) - À 300 DPI : fera 25 × 17 cm imprimée (haute densité, image nette)

    1.3 Formats de fichiers

    JPEG (.jpg) - Compression : avec perte - Avantages : fichiers légers, universel, idéal pour le web - Inconvénients : perte de qualité à chaque sauvegarde, pas de transparence - Usage : photographies, web, réseaux sociaux

    PNG (.png) - Compression : sans perte - Avantages : transparence possible, qualité préservée, bon pour les graphiques - Inconvénients : fichiers plus lourds que JPEG - Usage : logos, illustrations avec transparence, graphiques web

    PDF (.pdf) - Compression : variable - Avantages : préserve la mise en page, universel, peut contenir texte et images - Inconvénients : moins adapté pour l'édition d'images pures - Usage : documents à imprimer, portfolios, présentations professionnelles

    HEIC (.heic) - Compression : efficace avec bonne qualité - Avantages : fichiers légers, qualité élevée - Inconvénients : compatibilité limitée (standard Apple) - Usage : photos iPhone/iPad (à convertir en JPEG pour partage universel)

    Choisir le bon format - Photo pour Instagram → JPEG - Logo pour un site web → PNG - Affiche à imprimer → PDF ou JPEG haute résolution - Image avec fond transparent → PNG

    1.4 Poids des fichiers et optimisation

    Pourquoi optimiser? - Temps de chargement sur le web - Limites de taille sur les plateformes (ex: email, réseaux sociaux) - Espace de stockage

    Stratégies d'optimisation - Redimensionner l'image aux dimensions d'affichage réelles - Choisir le bon format (JPEG pour photos, PNG pour graphiques) - Ajuster la qualité de compression (80-85% pour JPEG est souvent invisible) - Utiliser des outils de compression en ligne si nécessaire

    Attention Gardez toujours une copie de votre fichier original haute résolution avant toute compression ou modification destructive.

    2. Canva : Votre atelier de retouche et de création

    Canva est une plateforme de design graphique en ligne accessible et puissante, parfaite pour retoucher vos images et créer des visuels professionnels.

    2.1 Pourquoi Canva?

    Accessibilité - Interface intuitive, courbe d'apprentissage rapide - Version gratuite très complète - Fonctionne dans le navigateur (pas d'installation) - Synchronisation cloud automatique

    Polyvalence - Retouche photo - Création graphique - Typographie - Modèles prédéfinis pour tous les formats (post Instagram, story, affiche, etc.)

    Limites du forfait gratuit - Accès limité à certains templates et éléments premium - Pas de fonctionnalités d'équipe avancées - Moins d'options de stockage cloud - Largement suffisant pour ce cours

    2.2 Interface et outils de base

    Structure de l'interface - Panneau latéral gauche : templates, éléments, uploads, texte, photos - Zone de travail centrale : votre canvas - Barre supérieure : fichier, redimensionner, partager, télécharger - Panneau de droite : propriétés de l'élément sélectionné

    Outils de retouche photo essentiels

    Ajustements de base - Luminosité : éclaircir/assombrir l'image - Contraste : accentuer la différence entre zones claires et sombres - Saturation : intensifier ou désaturer les couleurs - Netteté : rendre les détails plus précis

    Effets et filtres - Filtres prédéfinis (style Instagram) - Vignette : assombrir les bords pour diriger le regard - Flou : créer de la profondeur de champ - Grain : ajouter une texture film

    Corrections avancées - Balance des blancs : corriger la température de couleur - Teinte/Saturation : ajuster des couleurs spécifiques - Exposition : récupérer des détails dans les ombres/hautes lumières - Ombres/Hautes lumières : contrôle précis des zones claires et sombres

    2.3 Workflow de retouche recommandé

    Étape 1 : Import et recadrage - Importez votre image (glisser-déposer ou "Importer des fichiers") - Recadrez si nécessaire pour améliorer la composition - Vérifiez l'application de la règle des tiers

    Étape 2 : Corrections techniques - Ajustez l'exposition si l'image est trop claire ou trop sombre - Corrigez la balance des blancs si les couleurs semblent déséquilibrées - Réglez le contraste pour définir l'atmosphère

    Étape 3 : Améliorations créatives - Saturez légèrement pour vivifier les couleurs (attention aux excès!) - Ajoutez de la netteté avec parcimonie - Testez les filtres pour trouver une ambiance

    Étape 4 : Touches finales - Vignette subtile pour diriger le regard - Grain léger pour un effet cinématographique (optionnel)

    Règle d'or de la retouche

    "Less is more" — La retouche doit être invisible. Si le spectateur remarque la retouche avant l'image, c'est qu'on est allé trop loin.

    2.4 Modèles et dimensions

    Formats prédéfinis utiles - Post Instagram : 1080 × 1080 px (carré) - Story Instagram/Facebook : 1080 × 1920 px (vertical) - Post Facebook : 1200 × 630 px (horizontal) - Affiche : A4, A3 (selon besoin) - Présentation : 1920 × 1080 px

    Astuce Utilisez "Redimensionner" pour adapter un design existant à un nouveau format automatiquement.

    2.5 Export et sauvegarde

    Télécharger votre travail - Bouton "Partager" → "Télécharger" - Choisissez le format : PNG (qualité maximale, avec transparence), JPEG (plus léger), PDF

    Qualité d'export - Pour le web : JPEG qualité standard suffit - Pour l'impression : PNG ou PDF en haute qualité

    Organisation - Nommez vos fichiers de manière claire : projet1_affiche_finale.png - Gardez vos designs dans Canva pour y revenir (sauvegarde automatique)

    3. Typographie : Le texte comme élément visuel

    La typographie n'est pas qu'une question de lisibilité. C'est un outil narratif puissant qui influence la perception de votre message.

    3.1 Anatomie et vocabulaire typographique

    Familles de polices

    Serif (avec empattements) - Petites extensions aux extrémités des lettres - Effet : traditionnel, sérieux, élégant, littéraire - Exemples : Times New Roman, Garamond, Georgia - Usage : textes longs imprimés, journaux, marques classiques

    Sans-serif (sans empattements) - Lettres épurées, lignes nettes - Effet : moderne, direct, minimaliste, accessible - Exemples : Arial, Helvetica, Roboto - Usage : web, signalétique, marques tech, lisibilité écran

    Script (manuscrites) - Imitent l'écriture à la main - Effet : élégance, formalité (calligraphie) ou décontraction (manuscrite) - Exemples : Brush Script, Pacifico - Usage : invitations, logos féminins, accents décoratifs

    Display (décoratives) - Conçues pour attirer l'attention - Effet : expressif, unique, thématique - Exemples : Impact, Cooper Black - Usage : titres, affiches, courtes phrases d'accroche

    Termes clés - Graisse : épaisseur du trait (light, regular, bold, black) - Corps : taille de la police (en points : pt) - Interlignage : espacement vertical entre les lignes - Tracking/Crénage : espacement entre les lettres - Casse : majuscules, minuscules, capitales

    3.2 Psychologie de la typographie

    Le choix de police communique

    Une police serif classique pour une banque = confiance, stabilité Une police sans-serif épurée pour une startup tech = innovation, modernité Une police script élégante pour un mariage = romance, célébration Une police display audacieuse pour un concert rock = énergie, rébellion

    Règle fondamentale

    La police doit servir le message, pas le dominer. Si le lecteur remarque d'abord la police avant de lire le texte, c'est un problème.

    3.3 Hiérarchie typographique

    Pourquoi hiérarchiser? Pour guider l'œil du lecteur et structurer l'information du plus important au moins important.

    Niveaux de hiérarchie 1. Titre principal : le plus grand, le plus gras, capte l'attention en premier 2. Sous-titres : taille moyenne, structurent le contenu 3. Corps de texte : taille standard, lisibilité maximale 4. Texte secondaire : petit, informations complémentaires (légendes, crédits)

    Créer la hiérarchie - Taille : différence notable entre niveaux (ratio de 1,5 à 2×) - Graisse : bold pour titres, regular pour texte - Couleur : contraste pour les titres, gris pour le secondaire - Espacement : plus d'espace autour des titres

    Exemple de hiérarchie - Titre : 48 pt, gras, noir - Sous-titre : 24 pt, semi-gras, noir - Texte : 16 pt, regular, noir - Légende : 12 pt, regular, gris foncé

    3.4 Règles de lisibilité

    Pour les textes longs - Maximum 60-75 caractères par ligne - Interlignage généreux (1,4 à 1,6× la taille du texte) - Contraste élevé texte/fond (noir sur blanc ou l'inverse) - Éviter les polices trop décoratives

    Pour les titres et courts textes - Plus de liberté créative permise - Contraste visuel avec l'image de fond - Utilisation possible de polices expressives

    Couleur du texte sur image - Problème : lisibilité compromise si contraste insuffisant - Solutions : - Assombrir ou éclaircir l'image localement - Ajouter un arrière-plan semi-transparent sous le texte - Utiliser un contour ou une ombre portée - Placer le texte dans une zone naturellement claire ou sombre de l'image

    3.5 Harmonies typographiques

    Combiner des polices

    Règle de base Maximum 2-3 polices différentes dans un design. Au-delà, ça devient chaotique.

    Combinaisons efficaces - Serif + Sans-serif : classique et équilibré - Titre en serif élégant + corps en sans-serif lisible - Ou l'inverse : titre moderne sans-serif + corps serif traditionnel

    • Contraste de graisse : même police, différentes graisses
    • Titre en bold + corps en regular

    • Cohésion garantie

    • Police display pour titre + police neutre pour texte

    • Créativité dans le titre, sobriété dans le corps

    Éviter - Deux polices serif similaires (confusion) - Deux polices décoratives ensemble (trop chargé) - Polices qui se ressemblent trop sans être identiques (incohérence)

    Tester les combinaisons Canva propose des paires de polices harmonieuses dans ses modèles. Observez et analysez ces combinaisons pour comprendre ce qui fonctionne.

    3.6 Typographie et composition

    Le texte comme élément de composition

    Le texte n'est pas juste ajouté sur l'image : il fait partie de la composition visuelle.

    Placement stratégique - Respectez la règle des tiers pour positionner les blocs de texte - Suivez les lignes directrices de l'image - Équilibrez les masses visuelles (texte lourd = contrepoids visuel)

    Alignement - Gauche : naturel, lisible, informel - Centré : formel, équilibré, titres - Droite : rare, créatif, directionnel - Justifié : dense, formel, blocs de texte

    Respiration Laissez de l'espace autour du texte. Un texte trop collé aux bords ou à d'autres éléments étouffe le design.

    3.7 Erreurs courantes à éviter

    Trop de polices Limite-toi à 2-3 polices maximum. Chaque police supplémentaire dilue l'impact visuel.

    Mauvais contraste Texte gris clair sur fond gris = illisible. Teste toujours la lisibilité, surtout sur mobile.

    Texte trop petit Sur le web ou pour l'impression, assure-toi que le texte reste confortable à lire (minimum 12-14 pt pour le corps de texte).

    Excès d'effets Ombres portées, contours, dégradés : un seul suffit. L'accumulation d'effets alourdit le design.

    Ignorer la hiérarchie Si tout est gros et gras, rien n'est important. La hiérarchie guide le regard.

    All caps en excès LES MAJUSCULES SONT PLUS DIFFICILES À LIRE. Réserve-les aux titres courts et percutants.

    4. Synthèse : De la capture à la création

    Vous avez maintenant les outils pour : 1. Comprendre la nature technique de vos images (résolution, formats) 2. Retoucher vos photos pour sublimer votre intention initiale (Canva) 3. Ajouter du texte qui enrichit et structure votre message (typographie)

    4.1 Workflow complet

    Avant l'ordinateur (Bloc 1) - Cadrer selon la règle des tiers - Gérer la lumière et le contraste - Choisir une palette de couleurs

    Sur l'ordinateur (Bloc 2) - Importer et recadrer si nécessaire - Retoucher : exposition, contraste, couleurs - Ajouter de la typographie si le projet le demande - Exporter dans le bon format pour l'usage prévu

    4.2 Intention avant technique

    Question centrale « Quelle émotion ou quel message est-ce que je veux transmettre? »

    Tous vos choix techniques — du cadrage initial à la typographie finale — doivent découler de cette intention.

    La technique au service de l'émotion - La retouche renforce l'atmosphère capturée sur le terrain - La typographie complète ou contraste avec l'image - Le format d'export assure que votre message atteigne votre audience dans les meilleures conditions

    Exercice pratique recommandé

    Projet : Affiche événementielle

    À partir d'une photo que vous avez prise durant le Bloc 1 : 1. Importez-la dans Canva (format affiche A4) 2. Retouchez-la pour renforcer son atmosphère 3. Ajoutez un titre d'événement fictif (concert, exposition, conférence) 4. Ajoutez les informations secondaires (date, lieu) 5. Créez une hiérarchie typographique claire 6. Exportez en PDF haute qualité

    Critères d'évaluation - Qualité de la retouche (subtile et intentionnelle) - Lisibilité du texte sur l'image - Hiérarchie typographique claire - Cohérence entre l'image et le message textuel - Respect des principes de composition du Bloc 1
