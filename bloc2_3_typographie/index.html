    3. Typographie : Le texte comme élément visuel

    La typographie n'est pas qu'une question de lisibilité. C'est un outil narratif puissant qui influence la perception de votre message.

    3.1 Anatomie et vocabulaire typographique

    Référence : Anatomie de la typographie

    Familles de polices

    Serif (avec empattements)

    • Petites extensions aux extrémités des lettres
    • Effet : traditionnel, sérieux, élégant, littéraire
    • Exemples : Times New Roman, Garamond, Georgia
    • Usage : textes longs imprimés, journaux, marques classiques

    Sans-serif (sans empattements)

    • Lettres épurées, lignes nettes
    • Effet : moderne, direct, minimaliste, accessible
    • Exemples : Arial, Helvetica, Roboto
    • Usage : web, signalétique, marques tech, lisibilité écran

    Script (manuscrites)

    • Imitent l'écriture à la main
    • Effet : élégance, formalité (calligraphie) ou décontraction (manuscrite)
    • Exemples : Brush Script, Pacifico
    • Usage : invitations, logos féminins, accents décoratifs

    Display (décoratives)

    • Conçues pour attirer l'attention
    • Effet : expressif, unique, thématique
    • Exemples : Impact, Cooper Black
    • Usage : titres, affiches, courtes phrases d'accroche

    Termes clés

    • Graisse : épaisseur du trait (light, regular, bold, black)
    • Corps : taille de la police (en points : pt)
    • Interlignage : espacement vertical entre les lignes
    • Tracking/Crénage : espacement entre les lettres
    • Casse : majuscules, minuscules, capitales

    3.2 Psychologie de la typographie

    Référence : Psychologie des polices

    Le choix de police communique

    • Une police serif classique pour une banque = confiance, stabilité
    • Une police sans-serif épurée pour une startup tech = innovation, modernité
    • Une police script élégante pour un mariage = romance, célébration
    • Une police display audacieuse pour un concert rock = énergie, rébellion

    Règle fondamentale

    La police doit servir le message, pas le dominer. Si le lecteur remarque d'abord la police avant de lire le texte, c'est un problème.

    Association police-industrie

    • Finance/Droit : serif classiques (confiance, tradition)
    • Technologie : sans-serif modernes (innovation, clarté)
    • Mode/Luxe : serif élégantes ou sans-serif minimalistes (sophistication)
    • Enfants : rondes, ludiques (amusement, accessibilité)
    • Écologie : organiques, naturelles (authenticité, nature)

    3.3 Hiérarchie typographique

    Référence : Hiérarchie visuelle

    Pourquoi hiérarchiser?

    Pour guider l'œil du lecteur et structurer l'information du plus important au moins important.

    Niveaux de hiérarchie

    1. Titre principal : le plus grand, le plus gras, capte l'attention en premier
    2. Sous-titres : taille moyenne, structurent le contenu
    3. Corps de texte : taille standard, lisibilité maximale
    4. Texte secondaire : petit, informations complémentaires (légendes, crédits)

    Créer la hiérarchie

    • Taille : différence notable entre niveaux (ratio de 1,5 à 2×)
    • Graisse : bold pour titres, regular pour texte
    • Couleur : contraste pour les titres, gris pour le secondaire
    • Espacement : plus d'espace autour des titres

    Exemple de hiérarchie

    • Titre : 48 pt, gras, noir
    • Sous-titre : 24 pt, semi-gras, noir
    • Texte : 16 pt, regular, noir
    • Légende : 12 pt, regular, gris foncé

    3.4 Règles de lisibilité

    Référence : Lisibilité et typographie web

    Pour les textes longs

    • Maximum 60-75 caractères par ligne
    • Interlignage généreux (1,4 à 1,6× la taille du texte)
    • Contraste élevé texte/fond (noir sur blanc ou l'inverse)
    • Éviter les polices trop décoratives

    Pour les titres et courts textes

    • Plus de liberté créative permise
    • Contraste visuel avec l'image de fond
    • Utilisation possible de polices expressives

    Couleur du texte sur image

    • Problème : lisibilité compromise si contraste insuffisant
    • Solutions :
    • Assombrir ou éclaircir l'image localement
    • Ajouter un arrière-plan semi-transparent sous le texte
    • Utiliser un contour ou une ombre portée
    • Placer le texte dans une zone naturellement claire ou sombre de l'image

    Contraste minimum

    • WCAG 2.1 recommande un ratio de contraste d'au moins 4,5:1 pour le texte normal
    • 3:1 pour les textes de grande taille (18pt et plus)
    • Utilisez des outils comme WebAIM Contrast Checker pour vérifier

    3.5 Harmonies typographiques

    Référence : Combiner des polices

    Combiner des polices

    Règle de base

    Maximum 2-3 polices différentes dans un design. Au-delà, ça devient chaotique.

    Combinaisons efficaces

    • Serif + Sans-serif : classique et équilibré
    • Titre en serif élégant + corps en sans-serif lisible
    • Ou l'inverse : titre moderne sans-serif + corps serif traditionnel

    • Contraste de graisse : même police, différentes graisses
    • Titre en bold + corps en regular

    • Cohésion garantie

    • Police display pour titre + police neutre pour texte

    • Créativité dans le titre, sobriété dans le corps

    Éviter

    • Deux polices serif similaires (confusion)
    • Deux polices décoratives ensemble (trop chargé)
    • Polices qui se ressemblent trop sans être identiques (incohérence)

    Tester les combinaisons

    Canva propose des paires de polices harmonieuses dans ses modèles. Observez et analysez ces combinaisons pour comprendre ce qui fonctionne.

    Ressources pour trouver des combinaisons

    • Google Fonts : paires suggérées
    • Font Pair : combinaisons Google Fonts
    • Typewolf : inspiration de sites réels

    3.6 Typographie et composition

    Le texte comme élément de composition

    Le texte n'est pas juste ajouté sur l'image : il fait partie de la composition visuelle.

    Placement stratégique

    • Respectez la règle des tiers pour positionner les blocs de texte
    • Suivez les lignes directrices de l'image
    • Équilibrez les masses visuelles (texte lourd = contrepoids visuel)

    Alignement

    • Gauche : naturel, lisible, informel
    • Centré : formel, équilibré, titres
    • Droite : rare, créatif, directionnel
    • Justifié : dense, formel, blocs de texte

    Respiration

    Laissez de l'espace autour du texte. Un texte trop collé aux bords ou à d'autres éléments étouffe le design.

    Principe de proximité

    Les éléments liés doivent être proches, les éléments distincts doivent être espacés. Cela aide le lecteur à comprendre les relations entre les informations.

    3.7 Erreurs courantes à éviter

    Trop de polices

    Limite-toi à 2-3 polices maximum. Chaque police supplémentaire dilue l'impact visuel.

    Mauvais contraste

    Texte gris clair sur fond gris = illisible. Teste toujours la lisibilité, surtout sur mobile.

    Texte trop petit

    Sur le web ou pour l'impression, assure-toi que le texte reste confortable à lire (minimum 12-14 pt pour le corps de texte).

    Excès d'effets

    Ombres portées, contours, dégradés : un seul suffit. L'accumulation d'effets alourdit le design.

    Ignorer la hiérarchie

    Si tout est gros et gras, rien n'est important. La hiérarchie guide le regard.

    All caps en excès

    LES MAJUSCULES SONT PLUS DIFFICILES À LIRE. Réserve-les aux titres courts et percutants.

    Mauvais interlignage

    • Trop serré : texte étouffant, difficile à suivre
    • Trop large : perte de cohésion entre les lignes
    • Règle générale : 1,4 à 1,6× la taille du texte

    Kerning inadéquat

    L'espacement entre certaines paires de lettres (comme "AV" ou "To") peut créer des trous visuels. Les bonnes polices corrigent cela automatiquement, mais restez vigilant.

    3.8 Typographie dans Canva : conseils pratiques

    Trouver les bonnes polices

    • Menu "Texte" → Bibliothèque de polices
    • Filtres : toutes, vos polices, téléchargées
    • Recherche par nom ou par style

    Polices recommandées pour débuter

    Serif

    • Playfair Display (élégant, titres)
    • Merriweather (lisible, corps de texte)
    • Libre Baskerville (classique, polyvalent)

    Sans-serif

    • Montserrat (moderne, populaire)
    • Open Sans (très lisible, web)
    • Raleway (élégant, titres)

    Display

    • Bebas Neue (impact, masculin)
    • Pacifico (décontracté, manuscrit)
    • Lobster (rétro, accrocheur)

    Modifier le texte dans Canva

    • Double-clic pour éditer le contenu
    • Barre d'outils supérieure : police, taille, couleur, alignement
    • Panneau de droite : espacement, effets, transparence
    • Effets de texte : ombre, contour, arrière-plan

    Texte courbe et effets spéciaux

    • Sélectionnez le texte → "Effets" → "Courbe"
    • Ajustez l'intensité avec le curseur
    • Utile pour logos, badges, designs circulaires

    Grille typographique

    • Utilisez les guides pour aligner vos textes
    • Respectez une grille invisible pour la cohérence
    • Les éléments alignés créent une impression de professionnalisme
