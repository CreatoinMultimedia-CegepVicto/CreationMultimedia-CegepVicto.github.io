    Bloc 2 : De l'image captée à l'image travaillée

    1. Comprendre l'image numérique

    Après avoir capté vos images sur le terrain en appliquant les principes de composition, lumière et couleur, il est temps de comprendre leur nature technique pour mieux les travailler.

    1.1 Pixels et résolution

    Référence : Comprendre les pixels et la résolution

    Qu'est-ce qu'un pixel?

    Le pixel (picture element) est la plus petite unité d'une image numérique. C'est un carré de couleur uniforme. Une image numérique est composée de milliers, voire de millions de pixels assemblés comme une mosaïque.

    Résolution d'image

    La résolution désigne le nombre de pixels contenus dans une image, exprimée en largeur × hauteur.

    • Exemple : 1920 × 1080 pixels = 2 073 600 pixels (environ 2 mégapixels)
    • Plus il y a de pixels, plus l'image peut être affichée en grand sans perte de qualité

    Résolution d'affichage (PPI/DPI)

    • PPI (Pixels Per Inch) : nombre de pixels par pouce pour les écrans
    • DPI (Dots Per Inch) : nombre de points par pouce pour l'impression
    • 72-96 PPI : standard pour l'affichage écran (web, réseaux sociaux)
    • 300 DPI : standard pour l'impression de qualité

    Règle pratique

    • Pour le web : privilégiez des dimensions adaptées (1080px de large pour Instagram, 1200px pour Facebook)
    • Pour l'impression : calculez vos dimensions en fonction de la taille finale (ex: 10cm × 15cm à 300 DPI = 1181 × 1772 pixels)

    1.2 Définition vs Résolution

    Référence : Définition et résolution

    Définition

    Nombre total de pixels dans l'image (largeur × hauteur)

    • Une image de 4000 × 3000 pixels a une définition de 12 mégapixels

    Résolution

    Densité de pixels dans un espace donné (PPI/DPI)

    • La même image de 12 mégapixels peut avoir différentes résolutions selon son usage

    Exemple concret

    Une photo de 3000 × 2000 pixels :

    • À 72 PPI : fera 106 × 71 cm à l'écran (basse densité, pixels visibles de près)
    • À 300 DPI : fera 25 × 17 cm imprimée (haute densité, image nette)

    1.3 Formats de fichiers

    Référence : Guide des formats d'image

    JPEG (.jpg)

    • Compression : avec perte
    • Avantages : fichiers légers, universel, idéal pour le web
    • Inconvénients : perte de qualité à chaque sauvegarde, pas de transparence
    • Usage : photographies, web, réseaux sociaux

    PNG (.png)

    • Compression : sans perte
    • Avantages : transparence possible, qualité préservée, bon pour les graphiques
    • Inconvénients : fichiers plus lourds que JPEG
    • Usage : logos, illustrations avec transparence, graphiques web

    PDF (.pdf)

    • Compression : variable
    • Avantages : préserve la mise en page, universel, peut contenir texte et images
    • Inconvénients : moins adapté pour l'édition d'images pures
    • Usage : documents à imprimer, portfolios, présentations professionnelles

    HEIC (.heic)

    • Compression : efficace avec bonne qualité
    • Avantages : fichiers légers, qualité élevée
    • Inconvénients : compatibilité limitée (standard Apple)
    • Usage : photos iPhone/iPad (à convertir en JPEG pour partage universel)

    Choisir le bon format

    • Photo pour Instagram → JPEG
    • Logo pour un site web → PNG
    • Affiche à imprimer → PDF ou JPEG haute résolution
    • Image avec fond transparent → PNG

    1.4 Poids des fichiers et optimisation

    Pourquoi optimiser?

    • Temps de chargement sur le web
    • Limites de taille sur les plateformes (ex: email, réseaux sociaux)
    • Espace de stockage

    Stratégies d'optimisation

    • Redimensionner l'image aux dimensions d'affichage réelles
    • Choisir le bon format (JPEG pour photos, PNG pour graphiques)
    • Ajuster la qualité de compression (80-85% pour JPEG est souvent invisible)
    • Utiliser des outils de compression en ligne si nécessaire

    Attention

    Gardez toujours une copie de votre fichier original haute résolution avant toute compression ou modification destructive.

    Outils de compression recommandés

    • TinyPNG : compression PNG et JPEG
    • Squoosh : compression et conversion de formats
    • Compressor.io : compression multi-formats
