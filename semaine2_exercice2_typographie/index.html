    Semaine 2 - Exercice 2 : Exploration typographique

    Objectif

    Comprendre l'impact des choix typographiques en expérimentant avec différentes polices, tailles et hiérarchies.

    Durée estimée

    20-25 minutes

    Contexte

    La typographie peut complètement transformer le message d'une image. Cet exercice vous permettra de constater par vous-même comment le choix de police, la taille et la hiérarchie influencent la perception.

    Matériel requis

    • Compte Canva (utilisez votre projet de l'exercice 1 ou créez-en un nouveau)
    • Une phrase courte à designer (voir ci-dessous)

    Partie 1 : Le pouvoir de la police (10 min)

    Phrase à utiliser

    Choisissez UNE de ces phrases : - "CONCERT DE JAZZ - Vendredi 20h" - "FESTIVAL GOURMAND - Ce weekend" - "EXPOSITION D'ART - Entrée gratuite" - "TOURNOI E-SPORT - Inscriptions ouvertes"

    Étape 1 : Créer 4 variations (8 min)

    Instructions : 1. Dans Canva, créez un nouveau projet "Post Instagram" (1080 × 1080 px) 2. Ajoutez un fond de couleur unie (choisissez dans "Éléments" → "Formes" → Carré) 3. Ajoutez votre phrase choisie avec l'outil "Texte" 4. Dupliquez votre page 3 fois (vous aurez 4 pages au total)

    Créez 4 versions différentes, chacune sur une page :

    Version 1 : Élégance classique - Police : Serif élégante (ex: Playfair Display, Libre Baskerville, Cormorant) - Couleur de fond : crème, beige pâle ou blanc cassé - Couleur du texte : noir ou brun foncé - Effet attendu : traditionnel, sophistiqué, formel

    Version 2 : Modernité épurée - Police : Sans-serif simple (ex: Montserrat, Raleway, Poppins) - Couleur de fond : blanc pur, gris clair ou noir - Couleur du texte : contraste élevé avec le fond - Effet attendu : contemporain, direct, minimaliste

    Version 3 : Énergie et impact - Police : Display audacieuse (ex: Bebas Neue, Anton, Impact) - Couleur de fond : couleur vive (rouge, orange, jaune) - Couleur du texte : blanc ou noir selon le fond - Effet attendu : dynamique, accrocheur, urgent

    Version 4 : Créativité manuscrite - Police : Script ou manuscrite (ex: Pacifico, Dancing Script, Lobster) - Couleur de fond : couleur douce (pastel, tons neutres) - Couleur du texte : complémentaire au fond - Effet attendu : personnel, chaleureux, décontracté

    Étape 2 : Analyse (2 min)

    Questions à vous poser pour chaque version : - Est-ce que la police correspond bien au type d'événement? - Quelle émotion cette version évoque-t-elle? - Laquelle attire le plus votre attention? - Laquelle semble la plus crédible pour l'événement?

    Notez mentalement : Certaines polices fonctionnent mieux que d'autres pour votre phrase. Pourquoi?

    Partie 2 : La hiérarchie typographique (10 min)

    Nouvelle phrase à utiliser

    Utilisez cette structure complète : - Titre : "FESTIVAL DE MUSIQUE" (ou un titre lié à votre phrase précédente) - Sous-titre : "3 jours d'émotions" - Informations : "Du 15 au 17 juin 2026" - Détails : "Parc Central - Entrée 25$"

    Étape 1 : Mauvaise hiérarchie (3 min)

    Instructions : 1. Créez une nouvelle page dans votre projet 2. Ajoutez les 4 niveaux de texte 3. Mettez TOUT au même format : - Même police - Même taille (ex: 36 pt pour tout) - Même graisse (regular partout) - Même couleur

    Observation : C'est illisible, confus. On ne sait pas par où commencer la lecture.

    Étape 2 : Bonne hiérarchie (5 min)

    Instructions : 1. Dupliquez votre page précédente 2. Créez une hiérarchie claire :

    Titre (niveau 1) - Police : Gras, sans-serif ou display - Taille : 72-80 pt - Couleur : noir ou couleur vive - Position : en haut, centré

    Sous-titre (niveau 2) - Police : Peut être différente mais complémentaire, ou même police en plus léger - Taille : 36-42 pt - Couleur : noir ou gris foncé - Position : juste sous le titre

    Informations (niveau 3) - Police : Regular, claire - Taille : 24-28 pt - Couleur : noir - Position : milieu de page

    Détails (niveau 4) - Police : Light ou regular - Taille : 16-20 pt - Couleur : gris moyen - Position : en bas

    Étape 3 : Comparaison (2 min)

    Comparez vos deux pages : - Dans la première, tous les éléments se battent pour l'attention → chaos - Dans la seconde, l'œil sait où aller en premier → clarté

    Principe clé : La hiérarchie guide le regard du lecteur du plus important au moins important.

    Partie 3 : Combinaison de polices (5 min)

    Exercice : Trouver des paires harmonieuses

    Instructions : 1. Créez une nouvelle page 2. Écrivez un titre avec une police (ex: "GRANDE OUVERTURE") 3. Écrivez un paragraphe court en dessous avec une autre police (ex: "Découvrez notre nouvelle boutique ce samedi. Offres spéciales toute la journée!")

    Testez 3 combinaisons :

    Combinaison 1 : Serif + Sans-serif - Titre : Police serif (Playfair Display) - Corps : Police sans-serif (Open Sans) - Résultat : Classique et équilibré

    Combinaison 2 : Display + Sans-serif neutre - Titre : Police display impactante (Bebas Neue) - Corps : Police sans-serif sobre (Roboto) - Résultat : Moderne et dynamique

    Combinaison 3 : Sans-serif bold + Sans-serif light (même famille) - Titre : Montserrat Bold - Corps : Montserrat Light - Résultat : Cohésion totale, minimaliste

    À éviter (créez un exemple pour voir) : - Deux polices serif similaires - Deux polices décoratives ensemble - Plus de 3 polices différentes dans un seul design

    Défis bonus (si vous avez le temps)

    Défi 1 : Typographie expressive

    Prenez le mot "SILENCE" et faites-le crier visuellement en jouant uniquement avec : - La taille (très grande) - La graisse (très grasse) - La couleur (rouge vif) - L'espacement (très serré)

    Constat : Le contenu du mot et son apparence se contredisent. C'est puissant!

    Défi 2 : Lisibilité sur image

    1. Importez une photo de l'exercice 1
    2. Ajoutez le texte "AVENTURE" en blanc
    3. Testez 3 solutions pour améliorer la lisibilité :
    4. Ajouter un rectangle semi-transparent noir sous le texte
    5. Ajouter une ombre portée au texte
    6. Assombrir une zone de l'image localement

    Quelle solution fonctionne le mieux?

    Points à retenir

    Psychologie des polices

    • Serif = tradition, confiance, sérieux
    • Sans-serif = modernité, simplicité, clarté
    • Script = élégance, personnalité, créativité
    • Display = impact, unicité, expression

    Règles d'or

    1. Maximum 2-3 polices par design
    2. Créer une hiérarchie claire (3-4 niveaux maximum)
    3. Le contraste est votre ami (taille, graisse, couleur)
    4. La lisibilité avant tout

    Hiérarchie efficace

    • Différences notables : ratio de 1,5 à 2× entre les niveaux
    • Espacement généreux : plus d'espace autour des éléments importants
    • Cohérence : même style de hiérarchie dans tout le projet

    Auto-évaluation

    Après cet exercice, je comprends : - [ ] Comment une police change complètement l'atmosphère d'un message - [ ] L'importance de la hiérarchie typographique - [ ] Comment combiner 2-3 polices harmonieusement - [ ] La différence entre serif, sans-serif, script et display - [ ] Comment assurer la lisibilité du texte sur une image

    Réflexion personnelle : Quelle combinaison avez-vous préférée et pourquoi? Notez-la pour l'utiliser dans vos futurs projets.

    Ressources pour aller plus loin

    Inspiration de combinaisons : - Google Fonts : regardez les paires suggérées - Font Pair : combinaisons populaires - Analysez les affiches que vous voyez : quelles polices sont utilisées ensemble?

    Exercice d'observation : Cette semaine, photographiez 3 affiches ou designs dans la rue. Analysez : - Combien de polices sont utilisées? - Quelle est la hiérarchie? - Est-ce que les choix typographiques correspondent au message?
