    2. Canva : Votre atelier de retouche et de création

    Canva est une plateforme de design graphique en ligne accessible et puissante, parfaite pour retoucher vos images et créer des visuels professionnels.

    2.1 Pourquoi Canva?

    Référence : Canva pour l'éducation

    Accessibilité

    • Interface intuitive, courbe d'apprentissage rapide
    • Version gratuite très complète
    • Fonctionne dans le navigateur (pas d'installation)
    • Synchronisation cloud automatique

    Polyvalence

    • Retouche photo
    • Création graphique
    • Typographie
    • Modèles prédéfinis pour tous les formats (post Instagram, story, affiche, etc.)

    Limites du forfait gratuit

    • Accès limité à certains templates et éléments premium
    • Pas de fonctionnalités d'équipe avancées
    • Moins d'options de stockage cloud
    • Largement suffisant pour ce cours

    2.2 Interface et outils de base

    Référence : Guide de démarrage Canva

    Structure de l'interface

    • Panneau latéral gauche : templates, éléments, uploads, texte, photos
    • Zone de travail centrale : votre canvas
    • Barre supérieure : fichier, redimensionner, partager, télécharger
    • Panneau de droite : propriétés de l'élément sélectionné

    Outils de retouche photo essentiels

    Ajustements de base

    • Luminosité : éclaircir/assombrir l'image
    • Contraste : accentuer la différence entre zones claires et sombres
    • Saturation : intensifier ou désaturer les couleurs
    • Netteté : rendre les détails plus précis

    Effets et filtres

    • Filtres prédéfinis (style Instagram)
    • Vignette : assombrir les bords pour diriger le regard
    • Flou : créer de la profondeur de champ
    • Grain : ajouter une texture film

    Corrections avancées

    • Balance des blancs : corriger la température de couleur
    • Teinte/Saturation : ajuster des couleurs spécifiques
    • Exposition : récupérer des détails dans les ombres/hautes lumières
    • Ombres/Hautes lumières : contrôle précis des zones claires et sombres

    2.3 Workflow de retouche recommandé

    Étape 1 : Import et recadrage

    • Importez votre image (glisser-déposer ou "Importer des fichiers")
    • Recadrez si nécessaire pour améliorer la composition
    • Vérifiez l'application de la règle des tiers

    Étape 2 : Corrections techniques

    • Ajustez l'exposition si l'image est trop claire ou trop sombre
    • Corrigez la balance des blancs si les couleurs semblent déséquilibrées
    • Réglez le contraste pour définir l'atmosphère

    Étape 3 : Améliorations créatives

    • Saturez légèrement pour vivifier les couleurs (attention aux excès!)
    • Ajoutez de la netteté avec parcimonie
    • Testez les filtres pour trouver une ambiance

    Étape 4 : Touches finales

    • Vignette subtile pour diriger le regard
    • Grain léger pour un effet cinématographique (optionnel)

    Règle d'or de la retouche

    "Less is more" — La retouche doit être invisible. Si le spectateur remarque la retouche avant l'image, c'est qu'on est allé trop loin.

    2.4 Modèles et dimensions

    Référence : Dimensions optimales pour plusieurs supports

    Formats prédéfinis utiles

    • Post Instagram : 1080 × 1080 px (carré)
    • Story Instagram/Facebook : 1080 × 1920 px (vertical)
    • Post Facebook : 1200 × 630 px (horizontal)
    • Bannière YouTube : 2560 × 1440 px
    • Affiche : A4 (21 × 29,7 cm), A3 (29,7 × 42 cm)
    • Présentation : 1920 × 1080 px (16:9)

    Astuce

    Utilisez "Redimensionner" pour adapter un design existant à un nouveau format automatiquement (fonctionnalité parfois limitée en version gratuite, mais les dimensions de base restent accessibles).

    Créer une dimension personnalisée

    • Cliquez sur "Créer un design"
    • Sélectionnez "Dimensions personnalisées"
    • Entrez la largeur et la hauteur souhaitées
    • Choisissez l'unité (px, cm, mm, in)

    2.5 Export et sauvegarde

    Télécharger votre travail

    • Bouton "Partager" → "Télécharger"
    • Choisissez le format : PNG (qualité maximale, avec transparence), JPEG (plus léger), PDF

    Qualité d'export

    • Pour le web : JPEG qualité standard suffit
    • Pour l'impression : PNG ou PDF en haute qualité
    • Pour la transparence : PNG uniquement

    Organisation

    • Nommez vos fichiers de manière claire : projet1_affiche_finale.png
    • Gardez vos designs dans Canva pour y revenir (sauvegarde automatique)
    • Créez des dossiers dans Canva pour organiser vos projets

    Partage et collaboration

    • Possibilité de générer un lien de partage
    • Exportation en plusieurs formats simultanés
    • Option d'impression directe via Canva Print (service payant)

    2.6 Raccourcis clavier utiles

    • Ctrl/Cmd + Z : Annuler
    • Ctrl/Cmd + Y : Rétablir
    • Ctrl/Cmd + D : Dupliquer l'élément sélectionné
    • Ctrl/Cmd + G : Grouper les éléments
    • Ctrl/Cmd + Shift + G : Dégrouper
    • Alt/Option + Glisser : Dupliquer en déplaçant
    • Shift + Glisser : Contraindre les proportions
    • R : Afficher les règles
    • Delete/Backspace : Supprimer l'élément sélectionné

    2.7 Astuces et bonnes pratiques

    Gestion des calques

    • Utilisez le panneau "Position" pour gérer l'ordre des éléments
    • "Premier plan" / "Arrière-plan" pour déplacer rapidement
    • Verrouillez les éléments que vous ne voulez plus modifier

    Utilisation des grilles

    • Activez la grille pour aligner précisément vos éléments
    • Utilisez les guides intelligents qui apparaissent lors du déplacement

    Bibliothèque d'éléments

    • Photos : banque d'images gratuites intégrée (millions d'images)
    • Éléments graphiques : formes, lignes, icônes
    • Attention aux éléments premium (couronne sur l'icône)

    Templates

    • Explorez les templates pour vous inspirer
    • Personnalisez-les complètement : couleurs, polices, images
    • Un template est un point de départ, pas une fin en soi
