    Semaine 2 - Exercice 1 : Prise en main de Canva

    Objectif

    Se familiariser avec l'interface de Canva et maîtriser la séquence complète d'opération : création, importation, manipulation, exportation.

    Durée estimée

    20-25 minutes

    Contexte

    Cet exercice vous permet d'explorer les outils de base de Canva sans pression créative. L'objectif est de comprendre le workflow technique avant de créer vos propres designs.

    Étapes de l'exercice

    1. Création du projet (2 min)

    Actions à réaliser : - Ouvrez Canva et connectez-vous (ou créez un compte gratuit) - Cliquez sur "Créer un design" - Choisissez "Post Instagram" (1080 × 1080 px) - Familiarisez-vous avec l'interface : panneau gauche, zone de travail, panneau droit

    Vérification : Vous devriez voir une zone de travail carrée vide au centre de votre écran.

    2. Importation d'une image (3 min)

    Actions à réaliser : - Dans le panneau de gauche, cliquez sur "Importer" - Glissez-déposez une photo de votre choix (ou cliquez pour parcourir vos fichiers) - Cliquez sur l'image importée pour l'ajouter à votre canvas - Redimensionnez-la pour qu'elle remplisse tout le cadre (glissez les coins)

    Vérification : Votre image devrait couvrir l'ensemble du canvas carré.

    3. Exploration des outils de retouche (10 min)

    Actions à réaliser :

    A. Recadrage - Cliquez sur votre image - Trouvez l'outil "Recadrer" dans la barre supérieure - Testez différents recadrages - Validez un recadrage qui améliore la composition

    B. Ajustements de base - Cliquez sur "Modifier l'image" (dans la barre supérieure ou le panneau de droite) - Explorez chaque ajustement et notez son effet : - Luminosité : déplacez le curseur de -100 à +100 - Contraste : testez l'intensification et l'atténuation - Saturation : observez l'effet sur les couleurs - Netteté : augmentez légèrement (attention aux excès!)

    C. Filtres - Dans "Modifier l'image", trouvez la section "Filtres" - Testez au moins 3 filtres différents - Observez comment ils changent l'atmosphère de l'image - Désactivez tous les filtres pour revenir à l'original

    D. Effets - Trouvez la section "Effets" dans "Modifier l'image" - Testez : - Vignette : ajoutez une légère vignette (curseur vers 20-30) - Flou : observez l'effet (mais ne l'appliquez pas complètement) - Grain : ajoutez un léger grain pour un effet cinéma

    Vérification : Vous devriez avoir appliqué au moins quelques ajustements visibles à votre image.

    4. Ajout d'un élément graphique (3 min)

    Actions à réaliser : - Dans le panneau de gauche, cliquez sur "Éléments" - Cherchez "cercle" dans la barre de recherche - Ajoutez un cercle de couleur à votre design - Redimensionnez-le et placez-le quelque part dans l'image - Dans le panneau de droite, ajustez : - Couleur : choisissez une couleur qui contraste ou complète l'image - Transparence : réduisez l'opacité à environ 50-70%

    Vérification : Vous avez un cercle semi-transparent superposé à votre image.

    5. Gestion des calques (2 min)

    Actions à réaliser : - Cliquez sur votre cercle - Trouvez les boutons "Position" dans la barre supérieure - Testez : - "Arrière-plan" (le cercle passe derrière l'image) - "Premier plan" (le cercle revient devant) - "Avancer" et "Reculer" pour un contrôle précis - Expérimentez avec l'ordre des éléments

    Vérification : Vous comprenez comment gérer la superposition des éléments.

    6. Duplication et variation (3 min)

    Actions à réaliser : - Cliquez sur votre image modifiée - Utilisez Ctrl+D (ou Cmd+D sur Mac) pour la dupliquer - Une nouvelle page apparaît avec la même composition - Sur cette page 2, modifiez complètement les ajustements : - Changez le filtre ou les réglages de luminosité/contraste - Créez une ambiance totalement différente

    Vérification : Vous avez 2 pages dans votre projet, chacune avec une ambiance différente.

    7. Exportation (2 min)

    Actions à réaliser : - Cliquez sur "Partager" en haut à droite - Sélectionnez "Télécharger" - Choisissez le format : PNG (pour la meilleure qualité) - Cochez "Toutes les pages" pour exporter les 2 versions - Cliquez sur "Télécharger" - Vérifiez que les fichiers sont bien dans votre dossier Téléchargements

    Vérification : Vous avez 2 fichiers PNG téléchargés sur votre ordinateur.

    Points à retenir

    Workflow de base Canva

    1. Créer → Choisir un format ou dimension personnalisée
    2. Importer → Ajouter vos images
    3. Modifier → Ajustements, filtres, effets
    4. Composer → Ajouter éléments graphiques et texte
    5. Exporter → Télécharger dans le bon format

    Raccourcis clés à mémoriser

    • Ctrl/Cmd + Z : Annuler
    • Ctrl/Cmd + Y : Rétablir
    • Ctrl/Cmd + D : Dupliquer
    • Delete : Supprimer l'élément sélectionné

    Organisation

    • Canva sauvegarde automatiquement votre travail
    • Nommez votre projet de manière claire (cliquez sur le titre en haut)
    • Vous pouvez retrouver tous vos projets sur la page d'accueil de Canva

    Auto-évaluation

    Cochez les compétences acquises : - [ ] Je sais créer un nouveau projet dans Canva - [ ] Je sais importer mes propres images - [ ] Je comprends la différence entre luminosité, contraste et saturation - [ ] Je sais appliquer un filtre et l'ajuster - [ ] Je peux ajouter des éléments graphiques - [ ] Je comprends la gestion des calques (avant/arrière) - [ ] Je sais dupliquer une page pour créer des variations - [ ] Je peux exporter mon travail en PNG

    Si vous avez coché au moins 6 cases, vous êtes prêt pour les exercices suivants!

    Problèmes courants et solutions

    Mon image ne remplit pas tout le cadre → Glissez les coins de l'image pour la redimensionner, ou double-cliquez sur l'image pour la centrer et remplir le cadre automatiquement.

    Je ne trouve pas l'outil de retouche → Cliquez d'abord sur l'image pour la sélectionner, puis cherchez "Modifier l'image" dans la barre supérieure.

    Mes modifications sont trop extrêmes → Utilisez les curseurs avec parcimonie : restez entre -30 et +30 pour des ajustements subtils.

    Je ne vois pas le bouton "Position" → Assurez-vous qu'un élément graphique (pas l'image de fond) est sélectionné.

    Mon export est flou → Choisissez PNG au lieu de JPEG pour la meilleure qualité, et assurez-vous que votre image originale avait une résolution suffisante.
